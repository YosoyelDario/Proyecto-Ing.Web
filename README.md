# Sistema de gestión de consultas y listas de espera.
En el sistema de salud actual de la municipalidad de santo domingo, se cuenta con el problema de que los usuarios deben esperar meses para consultas o examenes debido a la sobredemanda y falta de organización.
Este proyecto actualmente busca :


Usuario Objetivo.
- Pacientes: Personas que necesitan agendar y seguir el estado de sus interconsultas.
- Administrador de salud: Personal encargado de gestionar la disponibilidad y priorización de pacientes.
